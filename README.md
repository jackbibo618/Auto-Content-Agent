# Auto-Content-Agent 🤖

A lightweight Multi-Agent workflow designed for automated content generation and platform-specific refinement. 

## 🌟 Features
- **Context-Aware Generation**: Utilizes long-chain reasoning to build article structures based on daily trends.
- **Multi-Agent Collaboration**: 
  - `Searcher Agent`: Gathers context.
  - `Writer Agent`: Generates the main draft.
  - `Reviewer Agent`: Adapts the tone for specific platforms (e.g., WeChat, Xiaohongshu) using self-refine prompts.
- **Low Cost & High Efficiency**: Reduces content creation time by 85%.
